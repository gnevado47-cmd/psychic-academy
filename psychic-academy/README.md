El proyecto consta de los siguientes archivos:
- [index.html](file:///C:/Users/grego/.gemini/antigravity/scratch/psychic-academy/index.html): Estructura del visor interactivo y del panel pedagógico.

## 🎯 Características Destacadas
- **Narrativa Envolvente**: Diseñada para transformar el aprendizaje técnico en un reto de decodificación.
- **Micro-decisiones y Gestión de MP**: Mecánicas claras para reducir el temor a equivocarse (parálisis por error).
- **Medidor de Confianza**: Barra visual interactiva que representa el avance del aula hacia el 100% de desbloqueo de identidad.
